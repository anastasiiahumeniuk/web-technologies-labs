Лабораторна робота №2

Пошук і перегляд фільмів через AJAX

Це фронтенд для лабораторної з client-server взаємодії. Сторінки отримують дані з FastAPI за допомогою чистого JavaScript `fetch` без `jQuery`.

Відкрити каталог

Що вже реалізовано

  • Каталог фільмів із завантаженням із бекенду
  • Пошук за назвою через AJAX-запити
  • Сторінка деталей фільму з параметром `id`
  • Перевірка JavaScript-коду через `npm run lint`

Основні сторінки

Перейдіть до потрібного розділу для перевірки функціоналу.

Технології

Frontend: HTML, SCSS, JavaScript

Backend: FastAPI

Запити: `fetch`, JSON, GET

Лінтер: ESLint
