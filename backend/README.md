## Огляд

**Movie Recommendation Service** — це сучасний веб-сервіс для перегляду фільмів і серіалів, з можливістю отримувати **розумні рекомендації** на основі ваших оцінок, жанрів і схожих користувачів.  
Мета — створити інтелектуальну платформу, що поєднує **ML-рекомендації**, **контекстний пошук** і **зручний інтерфейс**. Сервіс використовує **векторну базу даних** для ефективного контекстного пошуку на основі **ембеддингів**, що дозволяє знаходити фільми за змістом запиту, а не лише за ключовими словами.


---

## Функціонал

- **Каталог медіа** — пошук, сортування, фільтрація, детальні сторінки  
- **Рекомендації** — персоналізований список від ML-моделі  
- **Контекстуальний пошук** — розуміння запитів типу *“фільм про корабель з актором Леонардо ДіКапріо”*  
- **Система користувачів** — реєстрація, логін, профіль  
- **Оцінки та вподобання** — формують базу для рекомендацій  

---

## Технологічний Стек

| Компонент | Технологія | Опис |
| :--- | :--- | :--- |
| **Backend / API** | FastAPI | Реалізація бізнес-логіки, авторизації та API |
| **Frontend / UI** | React, TypeScript | Інтерактивний інтерфейс користувача |
| **Основна База Даних** | PostgreSQL | Зберігання фільмів, користувачів та оцінок |
| **Векторна База Даних** | Qdrant / ChromaDB / pgvector | Зберігання векторних представлень (ембеддингів) для контекстного пошуку |
| **ML-Модуль** | Python (scikit-learn, SentenceTransformers) | Генерація персональних рекомендацій та текстових ембеддингів |
| **Контейнеризація** | Docker, Docker Compose | Автоматизований запуск усіх сервісів |
